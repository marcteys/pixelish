##Controls

**Mouse**

 * Left click : Create a point on the shape
 * Right click : Delete a shape
 
**Arrow keys**
 * LEFT/RIGHT : Increase/Decrease cubes size
 * ENTER : Close a shape
 * UP : Reload randomly the images
 * DOWN : Toggle helpers";
